## 🔴 后端停止时

### 改进前:

## 📈 使用建议

**日常使用:**
1. 启动程序后,先看右上角心跳是否绿色 ✅
2. 点击启动引擎,观察徽章和消息 ✅
3. 有任何疑问,先看心跳和引擎状态 ✅

**问题排查:**
1. 心跳红色? → 后端可能停止了
2. 引擎徽章消失? → 引擎可能停止了
3. 心跳黄色? → 网络可能不稳定

**最佳实践:**
- 保持浏览器标签页打开
- 定期查看心跳状态
- 重要操作前确认心跳正常
- 使用手动关闭按钮管理消息
